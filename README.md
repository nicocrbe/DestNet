# DestNet - Social Network

Features:

- Create, update and delete posts
- Post details
- Likes and creation date
- Authentication
- Comments
- Search posts by tags/title and pagination
- Recommended posts by tags
- TODO: followers feature
- TODO: chat feature

Technologies:

- HTML, CSS, JavaScript (React)
- Redux
- Node.js (Express)
- MongoDB (Mongoose)
- Authentication with Bcrypt and JWT
- Socket.io
- Google Auth
- MaterialUI
- Backend testing with Jest and Postman

# Instructions:

# Client:
    npm install -f
    npm start

# Server:
    npm install
    npm run dev
